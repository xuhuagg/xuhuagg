建立博客的过程中遇到的坑

发表于 2017-08-25 | 分类于 hexo |
字数统计: 303
  • test安装完Node、git、hexo，后续的初始化过程建议在Git Bash的控制台，有的教程一下在系统控制台 ，一下又切换到Git Bash，第一次初始化hexo就被一个教程弄混了，hexo相关组件出了问题，最后只好重新安装。
  • 购买域名后绑定github pages后域名解析迟迟不生效。控制台ping我的个人域名没有响应， ping xuhuagg.github.io显示的是美国主机的地址。但我访问 xuhuagg.github.io 没有显示我的 github pages 页面，我当时就傻眼了，以为是系统延迟。 等到第二天还是访问不到。于是我重开了一个号还是遇到了一样的情况。百度了好久，发现原来是cn后缀的域名实名制之前会进入serverHold 状态，是2009年12月14日以后由CNNIC对CN域名新增加的一种域名状态，主要原因就是域名没有备案。域名在这个状态下，所有解析都无法生效，要取消这个状态，只有通过注册商，向CNNIC提交该域名的备案号，CNNIC会在三个工作日内审核并取消该状态。
